Project 1 – 5x5 Number Placement Game System

ABSTRACT
This document presents the 5x5 Number Placement Game System, a GUI-based puzzle game developed for CEN 4020. 
The system supports multi-level gameplay, sound feedback, undo functionality, and persistent game logging.


1. INTRODUCTION
The 5x5 Number Placement Game System expands an initial prototype into a fully interactive graphical application 
following agile development practices. Sprint 1 focuses on core gameplay and usability features.


2. SYSTEM OVERVIEW
The game allows players to place sequential numbers into a 5x5 grid and an expanded Level 2 board with an outer ring. 
The system enforces placement rules and provides real-time feedback.


3. FUNCTIONAL REQUIREMENTS
- Graphical board interaction
- Automatic next-number generation
- Sound feedback for valid and invalid actions
- Undo and reset features
- Game completion logging
- Level 2 expansion


4. NON-FUNCTIONAL REQUIREMENTS
- User-friendly interface
- Responsive system behavior
- Data persistence
